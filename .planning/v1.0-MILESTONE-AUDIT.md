## Cross-Phase Integration

### Pipeline Wiring

```
CLI (cli.py)
  ├── glma index → pipeline.py → walker → detector → parser → chunks → comments → store → writer
  │                                       → relationships (pass 2) → cross-file resolve (pass 3) → writer
  ├── glma query → formatter.py ← store.get_file_record / get_chunks / get_relationships
  │              → notebook.py (for .ipynb, bypasses store)
  ├── glma watch → watch.py → pipeline.run_index (incremental) → store + writer
  └── glma export → export.py ← store.get_indexed_files / get_chunks / get_relationships
                        → _format_export_file → generate_index_md → generate_relationships_md
```
