<action>
Add a new method to `LadybugStore` in `02-worktrees/glma/src/glma/db/ladybug_store.py`:

```python
def traverse_relationships(self, chunk_ids: list[str], max_depth: int = 1) -> list[dict]:
    """BFS traversal of relationships up to max_depth hops.

    Args:
        chunk_ids: Starting chunk IDs for traversal.
        max_depth: Maximum traversal depth (1 = direct relationships only).

    Returns:
        List of relationship dicts with added 'depth' field (1-indexed).
    """
```

Implementation:
1. Initialize `visited = set(chunk_ids)`, `queue = [(cid, 1) for cid in chunk_ids]`, `results = []`
2. BFS loop:
   - Pop `(current_id, depth)` from queue front
   - If `depth > max_depth`: continue
   - Get outgoing: `self.get_outgoing_relationships(current_id)`
   - Get incoming: `self.get_incoming_relationships(current_id)`
   - For each relationship:
     - Add `'depth': depth` to the dict
     - Add to results
     - Determine next_id: for outgoing use `target_id`, for incoming use `source_id`
     - Skip if next_id is empty or next_id in visited
     - Skip self-referential edges (source_id == target_id)
     - Add next_id to visited, append `(next_id, depth + 1)` to queue
3. Return results

Safety: max_depth is capped at 10 in the CLI command (this method itself doesn't enforce a cap — the CLI does).
</action>

<acceptance_criteria>
- `ladybug_store.py` contains `def traverse_relationships(self, chunk_ids, max_depth=1)`
- `traverse_relationships` returns list of dicts with `depth` field
- `traverse_relationships` with `max_depth=1` returns only direct relationships (equivalent to calling get_outgoing + get_incoming for each chunk)
- `traverse_relationships` with `max_depth=2` follows one additional hop from targets/sources
- Self-referential edges (source_id == target_id) are skipped during traversal
- No infinite loops on circular relationships (visited set prevents revisits)

Implementation updates:

1. **Notebook dispatch (new):** After resolving repo root and filepath:
   ```python
   if filepath.endswith('.ipynb'):

           raise typer.Exit(1)
       ```
