# glma-linux-kernel

Exploration of the Linux kernel codebase using AI agents for documentation and analysis.

## What This Does

This worktree contains notebooks and scripts for analyzing the Linux kernel source code.
The goal is to build an AI agent that can help document codebases in a way that is easy
to understand and navigate — using the Linux kernel as a test case.

## Getting Started

1. `cd 02-worktrees/linux_kernel`
2. `uv sync` (if venv not yet created)
3. Start coding with the notebooks in `02-development/`

## Dependencies
